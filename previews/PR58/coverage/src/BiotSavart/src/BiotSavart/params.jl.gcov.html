     222              : ## Mandatory keyword arguments
     223              : 
     224              : - `Γ::Real`: vortex circulation (assumed constant);
     225              : 
     226              : - `a::Real`: vortex core size (assumed constant);
     227              : 
     228              : - `α::Real`: Ewald splitting parameter (inverse length scale). One can set
     229              :   `α = Zero()` to efficiently disable long-range computations.
     230              : 
     231              : - `Ls::Union{Real, NTuple{3, Real}}`: domain period in each Cartesian direction.
     232              :   If a single value is passed (e.g. `Ls = 2π`), it is assumed that periods are

     235              :   One can set `Ls = Infinity()` to disable periodicity. This should be done in combination with `α = Zero()`.

     238              :   This parameter is not required if `α = Zero()`.
     239              : 
     240              : - `rcut`: cutoff distance for computation of short-range interactions.
     241              :   For performance and practical reasons, the cutoff distance must be less than half the cell
     242              :   unit size in each direction, i.e. `rcut < minimum(Ls) / 2`.
     243              :   This parameter is not required if `α = Zero()`.
     244              : 
     245              : ## Optional keyword arguments and their defaults
     246              : 
     247              : ### General
     248              : 
     249              : - `quadrature::StaticSizeQuadrature = GaussLegendre(3)`: quadrature rule for short- and
     250              :   long-range interactions. For example, if `quadrature = GaussLegendre(4)`, then 4 evaluations
     251              :   of the Biot–Savart integrand will be done for each filament segment.
     252              : 
     253              : ### Short-range interactions
     254              : 
     255              : - `backend_short::ShortRangeBackend`: backend used to compute
     256              :   short-range interactions. The default is `CellListsBackend(2)`, unless periodicity is
     257              :   disabled, in which case `NaiveShortRangeBackend()` is used.
     258              :   See [`ShortRangeBackend`](@ref) for a list of possible backends;
     259              : 
     260              : - `use_simd::Bool = true`: whether to use explicit SIMD during the computation of short-range
     261              :   interactions. This applies to the CPU implementation of short-range interactions in
     262              :   periodic domains only, where the SIMD implementation can accelerate the computation of `erfc(x)`.
     263              :   Usually there is no reason to disable this, other than to verify the accuracy or
     264              :   performance of the SIMD implementation.
     265              : 
     266              : ### Long-range interactions
     267              : 
     268              : - `backend_long::LongRangeBackend = NonuniformFFTsBackend()`: backend used to compute
     269              :   long-range interactions. See [`LongRangeBackend`](@ref) for a list of possible backends;
     270              : 
     271              : - `longrange_truncate_spherical = false`: if `true`, perform a spherical truncation in
     272              :   Fourier space, discarding all wavenumbers such that ``|\\bm{k}| > k_{\\text{max}}``.
     273              :   This is not recommended as it leads to precision loss, and should be used for testing only
     274              :   (in particular, for verifying error estimates which assume this kind of truncation).
     275              : 
     276              : ### Local self-induced velocity
     277              : 
     278              : - `Δ = 0.25`: coefficient appearing in the local self-induced velocity (LIA
     279              :   term), which depends on the vorticity profile at the vortex core.
     280              : 
     281              :   Some common values of `Δ` are:
     282              : 
     283              :   * `Δ = 0.25` for a constant vorticity profile (default);
     284              : 
     285              :   * `Δ = 0.5` for a hollow vortex;
