     311              : ## Difference between `callback` and `affect!`
     312              : 
     313              : The difference between the `callback(iter)` and `affect!(iter)` functions is the time at
     314              : which they are called:
     315              : 
     316              : - the `affect!` function is called *before* performing Biot-Savart computations from the

     331              : ``\bm{v}_{\text{f}}(\bm{x}, t)``.

     333              :

     347              :     external_velocity = (x⃗, t) -> Vec3(0.1 * sin(2 * x⃗.z), 0, 0)
     348              : 
     349              : One usually wants the external velocity to be divergence-free, to preserve the
     350              : incompressibility of the flow.
     351              : 
     352              : ### Notes on energy estimation
     353              : 
     354              : When setting an external velocity ``\bm{v}_{\text{f}}(\bm{x}, t)``, one may also want to
     355              : impose an external streamfunction field ``\bm{ψ}_{\text{f}}(\bm{x}, t)`` via the **`external_streamfunction`**

     369              :     +
     370              :     ∫ |\bm{v}_{\text{f}}|^2 \, \mathrm{d}^3\bm{x}
     371              :     \right],
     372              : \end{align*}
     373              : ```
     374              : 
     375              : where ``\bm{ψ}_{\text{BS}}`` is the streamfunction obtained from the Biot-Savart law.
     376              : Accordingly, when passing `external_streamfunction`, the streamfunction values attached to
     377              : vortex filaments (in `iter.ψs`) will actually contain
     378              : ``\bm{ψ}_{\text{BS}} + 2 \bm{ψ}_{\text{f}}`` (note the factor 2) to enable the
